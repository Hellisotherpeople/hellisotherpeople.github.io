What's up with this website?

My previous portfolio website was a mess. I used a bloated template which relied on tons of libraries for various "pretty" but ultimately superflous effects. A portfolio website shouldn't have to deal with bugs related to the ~5mb of javascript libraries that I'm importing for it. Given that this is a static site, I felt no need to use any JS. A clean, simple website like this is part of my attempt to move towards elegance and minimalism with my workflow.

Yes, with only plain HTML/CSS, a website can look clean, crisp, and can render on all devices properly. It's also going to be blazing fast, and won't waste the precious cache space of my readers.
